# InstaClone
Insta clone in edicational purposes. Firebase as backend and auth.

- Can login
- Can sign in
- Can create posts (image from gallery or photo)
- Can comment
- Can like
- Can follow
- Can dele posts
- Can add avatar image
- Can search users by nickname
